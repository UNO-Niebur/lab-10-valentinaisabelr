# Lab 10 – Data Visualization

## Purpose
This lab focuses on:
- working with real data
- creating visualizations
- interpreting results

Detailed instructions are provided in Canvas.

---

## Files

- `reaction_time_data.csv` - input dataset
- `main.py` - loads data and generates visualization
- `output.png` - graph image generated by the script
- `requirements.txt` - Python dependency list

---

## Requirements

Your program should:
- load the dataset
- select relevant data
- create a graph using Matplotlib
- save the graph as an image

---

## Output

- One graph image (e.g., output.png)
- Python script

---

## Setup

Install dependencies:

```bash
python3 -m pip install -r requirements.txt
```

Run the script:

```bash
python3 main.py
```

Expected console output:

```text
Graph saved to output.png
```
